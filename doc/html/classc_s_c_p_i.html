Detailed Description

cSCPI is a class to support servers and modules to generate their SCPI interface.

To keep cSCPI clean, it only decodes the commands from input and returns a pointer to cSCPIObject, which must then decide what to do, whether is it a query or a command and if necessary return an answer.

It holds all scpi commands in a qstandarditemmodel.

The commandlist can be exported in xml and imported with xml.
